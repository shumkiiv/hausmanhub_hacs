# История версий

## 0.2.0 — 2026-07-14

Первая опубликованная безопасная версия HASC.

- Два режима: «Только чтение» и «Проверка без изменений».
- Настройки, безопасное удаление и обновление проверены в пустой учебной
  среде Home Assistant.
- Диагностика показывает только девять общих чисел о доме без названий,
  показаний, адресов или ключей.
- Локальная страница с этими же девятью числами доступна только специальному
  пользователю Home Assistant для просмотра и не принимает команды.
- Добавлена понятная местная проверка перед публикацией.

Эта версия не управляет домом: не создаёт службы, устройства или команды, не
меняет климат и автоматики, не включает передачу команд и не выполняет ничего
напрямую.

Для установки нужен Home Assistant Core 2026.6.4 или новее.
